# **App Name**: Civisys Risk Manager

## Core Features:

- Dashboard: Provides a quick overview of key metrics, including active projects, reported incidents, and projects at risk.
- Project Management: Allows engineers to manage project details, including location, status, progress, responsible party, and timelines.
- Incident Reporting: Enables users to report incidents with severity levels and detailed descriptions, including status tracking (review, mitigation, reported).
- Document Management: Provides an interface for uploading, viewing, and downloading project-related documents.
- Reporting and Statistics: Generates interactive charts and allows exporting data in PDF/CSV formats, offering insights into total incidents and project completion rates.
- Risk Assessment AI Tool: Uses a generative AI tool to identify potential risks based on project descriptions and historical incident data and incorporate them to potential issues with this particular project, by querying existing database of documentation for potential overlaps of projects in same physical area, project timelines, engineering personnel. The AI uses a tool to determine what data to incorporate.
- Login: Provides a secure login page with email, password, 'Remember Me', and 'Forgot Password' options.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for buttons, headers, and active icons, representing stability and trust.
- Background color: Light gray (#F9FAFB) for the main background to provide a clean and neutral canvas.
- Accent color: Light blue (#3B82F6) for secondary backgrounds and hover effects, adding a subtle highlight.
- Font: 'PT Sans', a sans-serif font, will be used for headings and body text to ensure readability and a modern feel.
- Material Icons will be used for actions, states, and navigation, providing a consistent and recognizable visual language.
- A minimalist layout will prioritize data visualization and easy navigation, featuring a dashboard with a sidebar and top header. On smaller screens, the sidebar collapses into a top menu icon.
- Subtle animations will be used to enhance the user experience, such as animated gradients in progress bars and smooth transitions on hover.
